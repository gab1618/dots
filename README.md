# My dotfiles

Feel free to copy them, just keep in mind it won't just work out of the box.

Tip: use stow to install the dots

## Dependencies

* FiraCode Nerd Font
* Neovim + Lazy.nvim
* Kitty
* zsh + oh-my-zsh + powerlevel10k
* Tmux + tpm
* Hyprland, Hyprlock, Hypridle, Hyprpaper
* Walker + Elephant
* Quickshell
* Grim, slurp, wl-clipboard
* Btop
